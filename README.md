## Run Project
#### npm install (최초 1회 실행)
```
meteor npm install
```
#### run
```
meteor
```

## Project Structure
* Client : 화면 구성 모음
* docs : 정리한 문서 모음
* public : 사진 파일 등 모음
* server : 백엔드 관련
* lib : html HEAD, 라이브러리, 함수 등..
